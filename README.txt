Bonjour et bienvenu(e) sur le programme RobOwl Multithreading!

Ci-dessous quelques indications d'utilisation du programme.

Apres avoir lance RUN, l'interface graphique apparait.


		*** COMMENT UTILISER LE PROGRAMME ?
	
Rien de plus simple!

- Commencez par cliquer sur l'ecran. Un menu defilant apparait et vous permet de placer,
au noeud le plus proche du clic, un feu, de choisir son intensite, de placer une source
ou un robot.

- Il y a differents types de robot! Le robot roue sera le plus rapide sur route normale
(noire), le robot chenille se defendra mieux sur un terrain escarpe (marron) et le robot
patte est le meilleur sur terrain inonde (bleu). Le robot drone quant a lui, est capable
de survoler la carte tres rapidement, mais attention, il ne possede que 3 recharges d'eau!

- Apres avoir installe un scenario initial, cliquez sur PLAY, la simulation commence. A tout
moment vous pouvez ajouter un feu, le retirer, ajouter un robot, sans stopper la simulation.
ATTENTION, vous ne pouvez pas en revanche ajouter de source pendant une simulation...

- Le bouton PAUSE conserve la simulation dans l'etat ou elle est stopee. Le bouton STOP
supprime la simulation en cours.

- Le curseur sous la carte permet de regler la vitesse de deplacement des robots.

- Sur le cote de l'ecran, une legende vous informe en temps reel de la situation de la
simulation.

- En bas de l'ecran, une ligne vous indique les coordonnees des noeuds suvoles par la souris,
avec une indication sur son caractere de source ou l'intensite du feu qui s'y trouve.


		*** LA BARRE DE MENU

- FILE -> OPEN MAP permet de selectionner une carte, en format OSM ou TXT
          parmis vos fichiers et de le charger pour la simulation. Par defaut, la
          la carte de l'ISAE est chargee sur l'ecran.
			
       -> OPEN SIMULATION permet de charger une simulation deja effectuee afin de
          la continuer.
		   
       -> SAVE SIMULATION permet de sauvegarder la simulation en cours dans un fichier
          texte, reutilisable par la suite.
		   
- OPTIONS -> ALGORITHM permet de choisir quel algorithme vous souhaitez utiliser pour
             le calcul du plus court chemin (DJIKSTRA ou A*)
				
- HELP -> renvoie a ce README
	
- ABOUT -> ROBOWL vous donne un apercu des veritables robots pompiers
